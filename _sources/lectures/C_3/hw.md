1. Для заданий где просится генерировать матрицу, советую вначале задать свою матрицу над которой будите проверять вашу программу
2. При использовании `rand()` ограничьтесь небольшим распределением, к примеру, U(0,10) или U(0, 100) (т.е. равномерным распределением от 0 до 10 или до 100)
3. Вы не ограничены в памяти, так что можете создавать любое необходимое количество массивов

| № | Формулировка задания |
|---|----------------------|
| **1** | **Грубый поиск последнего вхождения подмассива** в массиве с одной ошибкой (один элемент найденного вхождения может отличаться). Массив заполняется случайными значениями с помощью `rand()`. Размеры задаются через `#define`. Вывести количество сравнений, позицию, массив и подмассив. |
| **2** | **Транспонирование матрицы.** Размеры матрицы задаются через `#define`. Матрица заполняется случайными значениями с помощью `rand()`. Вывести исходную и транспонированную матрицы, их размерности. |
| **3** | Для квадратной матрицы вычислить **сумму элементов главной и побочной диагоналей**. Определить, пересекаются ли эти диагонали в одном элементе, и учесть это при подсчёте. |
| **4** | Сортировка массива действительных чисел методом **шейкера** в порядке убывания. Массив заполняется случайными значениями с помощью `rand()`. Размер задаётся через `#define`. Вывести количество перестановок и сравнений, входной и выходной массивы. |
| **4** | Найти **сумму элементов каждой строки** и **каждого столбца** матрицы. Определить, в какой строке и в каком столбце сумма максимальна. Матрица заполняется случайными числами с помощью `rand()`. |
| **5** | Найти **номер столбца**, где **среднее значение элементов** минимально. Вывести этот столбец и его среднее значение. |
| **6** | **Грубый поиск первого вхождения подмассива** в массиве с **одной ошибкой** (один элемент найденного вхождения может отличаться). Массив заполняется случайными значениями с помощью `rand()`. Размеры задаются через `#define`. Вывести количество сравнений, позицию, массив и подмассив. |
| **7** | Сортировка массива действительных чисел **методом вставки** **в порядке возрастания**. Массив заполняется случайными значениями с помощью `rand()`. Размер задаётся через `#define`. Необходимо вывести количество перестановок и сравнений, а также входной и выходной массивы. |
| **8** | Заменить в матрице все отрицательные элементы их **модулями** (то есть сделать все элементы неотрицательными). Вывести количество заменённых элементов. |
| **9** | Сортировка массива действительных чисел методом **выбора** в порядке возрастания. Массив заполняется случайными значениями с помощью `rand()`. Размер задаётся через `#define`. Вывести количество перестановок и сравнений, входной и выходной массивы. |
| **10** | **Грубый алгоритм поиска первого вхождения подмассива** в массиве. Массив заполняется случайными значениями с помощью `rand()`. Размер массива и подмассива задаётся через `#define`. Вывести количество сравнений, позицию начала вхождения, сами массив и подмассив. |
| **11** | Определить, сколько строк матрицы содержат хотя бы один **отрицательный элемент**, и сколько столбцов — только **положительные элементы**. |
| **12** | Выполнить **сумму и разность двух матриц** одинакового размера. Матрицы заполняются случайными числами. Вывести обе исходные матрицы и результаты операций. |
| **13** | Сортировка массива действительных чисел методом **пузырька** в порядке убывания. Массив заполняется случайными значениями с помощью `rand()`. Размер задаётся через `#define`. Вывести количество перестановок и сравнений, входной и выходной массивы. |
| **14** | Найти **след матрицы** (сумму элементов главной диагонали) и **определить, является ли матрица диагональной** (все элементы вне главной диагонали равны нулю). |
| **15** | Сортировка массива действительных чисел методом **выбора** в порядке убывания. Массив заполняется случайными значениями с помощью `rand()`. Размер задаётся через `#define`. Вывести количество перестановок и сравнений, входной и выходной массивы. |
| **16** | Реализовать **умножение двух матриц**. Размеры матриц заданы через `#define`. Матрицы заполняются случайными значениями. Вывести исходные и результирующую матрицы. |
| **17** | Определить, сколько **нулевых элементов** содержит каждая строка и каждый столбец матрицы. Вывести номера строк и столбцов, в которых нулей больше всего. |
| **18** | Написать программу для **нахождения максимального и минимального элементов двумерного массива** и их индексов. Матрица заполняется случайными числами, размеры задаются через `#define`. |
| **19** | Сортировка массива действительных чисел **методом вставки** **в порядке убывания**. Массив заполняется случайными значениями с помощью `rand()`. Размер задаётся через `#define`. Вывести количество перестановок и сравнений, входной и выходной массивы. |
| **20** | Сортировка массива действительных чисел методом **пузырька** в порядке возрастания. Массив заполняется случайными значениями с помощью `rand()`. Размер задаётся через `#define`. Вывести количество перестановок и сравнений, входной и выходной массивы. |
| **21** | Найти и вывести **наибольший элемент каждой строки** и **наименьший элемент каждого столбца**. |
| **23** | Реализовать **сортировку массива строк** (символьных последовательностей) в алфавитном порядке. Строки вводятся пользователем или задаются статически. Вывести исходный и отсортированный массив строк. |
| **24** | Проверить, является ли квадратная матрица **магическим квадратом** (суммы элементов в каждой строке, столбце и диагоналях совпадают). |
| **25** | **Грубый алгоритм поиска последнего вхождения подмассива** в массиве. Массив и подмассив заполняются случайными значениями с помощью `rand()`. Размеры задаются через `#define`. Вывести количество сравнений, позицию начала вхождения, массив и подмассив. |
| **26** | **Грубый поиск первого вхождения подмассива** в массиве с **заданным пользователем числом ошибок**. Массив заполняется случайными значениями с помощью `rand()`. Размеры задаются через `#define`. Вывести количество сравнений, позицию, массив и подмассив. |
| **27** | Найти **транспонированную матрицу**, затем проверить, является ли исходная матрица **симметричной** (`A == A^T`). Матрица квадратная, размеры задаются через `#define`. |
| **28** | Выполнить **обмен строк** матрицы: первую строку поменять с последней, вторую — с предпоследней и т.д. Вывести исходную и преобразованную матрицы. |
| **29** | Выполнить **нормализацию матрицы**: каждый элемент разделить на максимальный элемент матрицы, чтобы все значения лежали в диапазоне `[0,1]`. |
